## 📂 변수(Variable)

#### 📌 변수란?
- 값을 저장하기 위한 메모리(RAM) 내의 기억장소
- 값(객체)이 저장된 메모리의 위치를 가리키는 참조(Reference)

#### 📌 파이썬의 변수
- 파이썬에서 변수 선언은 필수적이지 않다
- 필요한 시점에 변수를 만들어서 값을 지정하면 된다
- 변수에 저장된 값의 형(Type)에 따라 실행 시점에서 형(Type) 검사 ➝ **동적 타이핑(Dynamic Typing)**

#### 📌 변수 명명규칙
- 대소문자 구분
- 영문자 · 숫자 · 밑줄로 구성
- **낙타체** 사용

#### 📌 변수에 값을 저장하는 방법
- 변수 = 값
- 변수1, 변수2, 변수3 = 값1, 값2, 값3
- 변수1 = 변수2 = 변수3 = 값

```python
x = 10
a, b, c = 1, 2, 3
a = b = c = 5
```

#### 📌 x = 10의 의미?
- 변수 x (100번지) ➝ 객체 10
- **변수는 객체를 참조하는 이름표이고 실제 데이터는 객체 안에 저장된다**

#### 📌 변수를 삭제하는 방법
- del 변수명

```python
a = 100
del a
```

#### 📌 두 변수의 값 교환(Swap)
- 변수1, 변수2 = 변수2, 변수1

```python
a = 1
b = 2

a, b = b, a

print(a) ➝ 2
print(b) ➝ 1
```

## 📂 print() · id() 함수

#### 📌 print()
- print(변수) : 변수가 가리키는 값을 출력

```python
name = "홍길동"
age = 23

print(name, age) ➝ 홍길동 23

address = "서울시 강남구"

print(name + '은 ' + address + '에 삽니다') ➝ 홍길동은 서울시 강남구에 삽니다
print(name + "은 " + str(age) + "살입니다" ➝ 홍길동은 23살입니다

※ 숫자를 문자열과 연결하기 위해서는 숫자를 문자열 타입으로 변환하는 형변환 함수인 str() 사용
```

#### 📌 id()
- id(변수) : 변수가 가리키는 메모리의 주소 반환

```python
x = 10

id(x) ➝ 1630430528
```

## 📂 식별자(Identifier) · 예약어(Reserved Words)

#### 📌 식별자

- 변수 · 상수 · 함수 · 사용자 정의 타입 등에서 다른 것과 구분하기 위해 사용되는 이름

#### 📌 예약어

- 이미 용도가 정해져 있는 단어
- 변수 이름이나 함수 이름 등의 식별자로 사용할 수 없다
- ex) 'False', 'None', 'and', 'def', 'import', ...

## 📂 포맷 코드(Format Code)

#### 📌 포맷 코드 종류

| 포맷 코드 | 설명      |
| :-------: | :-------: |
| %s        | 문자열    |
| %c        | 문자 1개  |
| %d        | 정수      |
| %f        | 부동 소수 |
| %%        | % 자체    |

#### 📌 형식 : print("서식" % 값)

```python
print("성명 : %s" % name)
print("학년 : %d" % year)
print("학점 : %c" % grade)
print("평균 : %f" % average)
print("출석률 : %d%%" % rate)
print("%d, %f" % (total, average))
```

## 📂 자릿수 설정

```python
fTemp = 90
cTemp = (fTemp - 32) * 5 / 9

print(cTemp) ➝ 32.2222222222222
print("%f" % cTemp) ➝ 32.222222 ※ 소수점 이하 6자리가 디폴트 값
print("%.2f" % cTemp) ➝ 32.22 ※ 소수점 이하 2자리
print(format(cTemp, '10.2f')) ➝ _____32.22 ※ 10자릿수, 소수점 이하 2자리
```

## 📂 상수(Constant)

#### 📌 상수란?
- 변경되지 않는 값
- 파이썬에서는 별도의 상수가 존재하지 않는다
- 변수와 구분하기 위해 대문자로 사용할 뿐 나중에 상수의 값을 변경해도 오류가 발생하지 않는다

```python
PI = 3.141592
TAX_RATE = 0.35
```

## 📂 리터럴(Literal)

#### 📌 리터럴이란?

- 고정된 '값'
- 변수에 저장되는 '값'

#### 📌 리터럴의 종류

- 정수 리터럴
  - 0b로 시작하는 2진수
  - 0o로 시작하는 8진수
  - 0~9로 시작하는 10진수
  - 0x로 시작하는 16진수
- 실수 리터럴
  - 0.25, -3.14와 같이 소수점이 있는 실수이거나 e를 포함하는 지수 표현
- 문자 리터럴
  - 'A', 'B'와 같이 따옴표로 묶은 하나의 문자
- 문자열 리터럴
  - 'Python', "Python", '''Python''', """Python"""과 같이 따옴표로 묶은 일련의 문자
- 논리값 리터럴
  - True, False
- 특수 리터럴
  - 값(객체)이 없음
  - None

## 📂 이스케이프 문자(Escape Character)

#### 📌 이스케이프 문자란?
- 프로그래밍 할 때 사용할 수 있도록 미리 정의해둔 문자 조합

#### 📌 대표적인 이스케이프 문자

| 이스케이프 문자 | 출력되는 문자  |
| :-------------: | :------------: |
| \\\             | 백슬래시(\\)   |
| \\\'            | 작은 따옴표(') |
| \\\"            | 큰 따옴표(")   |
| \n              | 줄 바꿈        |
| \t              | 탭             |

#### 📌 특수문자 의미를 없애는 방법
- 첫 번째 따옴표 앞에 r 추가

```python
print("c:\info\name) ➝ print(r"c:\info\name)
```
