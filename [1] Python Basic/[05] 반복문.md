## 📂 for 문

#### 📌 for 문이란?
- 정해진 횟수만큼 문장 반복 수행

#### 📌 형식

```python
for 변수 in 범위 또는 리스트:
  반복할 문장
```

#### 📌 range() 함수
- 특정 범위의 정수 생성
- range(number) : 0부터 number-1까지의 정수
- range(start, stop) : start에서 stop-1까지의 정수
- range(start, stop, step) : start에서 stop-1까지 step 간격으로 정수 생성

## 📂 while 문

#### 📌 while 문이란?
- 정해진 조건에 따라 반복 수행
- 조건식을 먼저 확인한 후 참이면 문장 반복 수행

#### 📌 형식

```python
while 조건:
  반복문장
  증감식 ※ 증감식은 초기 값이 반드시 있어야 한다
```

## 📂 break 문

#### 📌 break 문이란?
- 반복문 내에서 의도적으로 제어 흐름을 중단하고 빠져나온다

```python
while True:
  print("반복 수행되는 문장입니다.")
  answer = input("종료하려면 x 입력 : ")
  
  if answer == 'x':
    break
```

## 📂 continue 문

#### 📌 continue 문이란?
- 반복문 수행 중 continue를 만나면 제어 흐름을 유지한 상태에서 코드의 실행만 건너뛴다

```python
x = 0

while x < 10:
  x += 1
  
  if x % 2 == 0: ※ 짝수이면 중단하고 다음 반복으로 넘어감
    continue
  print(x) ➝ 1
              3
              5
              7
              9
```
