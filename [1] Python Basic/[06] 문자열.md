## 📂 문자열(String)

#### 📌 문자열과 문자열의 연결
- '+' 연산자 사용

```python
name = "홍길동"
address = "서울"

print("name + "은 " + address + "에 삽니다") ➝ 홍길동은 서울에 삽니다
```

#### 📌 문자열과 숫자 연결
- 숫자를 문자열로 형변환 후 '+' 연산자로 연결

```python
r = 10
pi = 3.141592
area = r ** 2 * pi

print("원의 넓이는 " + str(area) + "입니다") ➝ 원의 넓이는 314.1592입니다
```

#### 📌 문자열 곱하기
- 곱하는 수 만큼 문자열을 반복

```python
a = "python"

print(a * 3) ➝ pythonpythonpython
```

#### 📌 문자열 인덱싱(Indexing)
- 인덱스로 문자의 위치를 나타내는 것

```python
a = "python"

p y t h o n
0 1 2 3 4 5

a[0] ➝ p
a[1] ➝ y
a[5] ➝ n
a[-1] ➝ n
```

#### 📌 문자열 슬라이싱(Slicing)
- 문자열 중에서 일부분을 추출하는 것
- 원본 문자열 중 일부를 복사해서 반환
- 문자열 원본 변수는 변경되지 않음

```python
string[0:n] ➝ 0부터 n-1번째까지의 문자열
string[n:] ➝ n부터 끝까지
string[:n] ➝ 0부터 n-1까지
string[:] ➝ 문자열 전체
```

#### 📌 in / not in
- 문자열 안에 특정 문자열이 포함되어 있는지 여부 확인 결과는 True / False

## 📂 문자열 관련 함수(메소드)

#### 📌 len()
- 문자열의 길이 반환

```python
a = "python"

print(len(a)) ➝ 6
```

#### 📌 count()
- 문자열 내에 들어 있는 특정 문자(열)의 개수 반환

```python
a = "programming"

print(a.count("m")) ➝ 2
```

#### 📌 find()
- 문자열 내에서 특정 문자열이 존재하는지 여부와 문자열의 시작 위치를 반환
- 존재하지 않는다면 -1을 반환

```python
a = "Data Crawling is fun"

print(a.find("fun")) ➝ 17
print(a.find("f")) ➝ 17
print(a.find("parsing")) ➝ -1
```

#### 📌 index()
- 문자열 내에서 특정 문자의 시작 위치를 반환
- 존재하지 않는 문자를 찾는다면 오류 발생 **※ find()와 다른 점**

#### 📌 split()
- 구분자를 기준으로 문자열을 나눔
- 결과를 리스트로 반환

```python
a = "Life is too short"

print(a.split()) ➝ ['Life', 'is', 'too', 'short']

b = "a:b:c:d"

print(b.split(":")) ➝ ['a', 'b', 'c', 'd']
```

#### 📌 replace()
- 전체 문자열에서 특정 문자열을 찾아 다른 문자열로 변경

```python
a = "Life is too short"

print(a.replace("Life", "Your leg")) ➝ Your leg is too short
```

#### 📌 join()
- 각 문자 사이에 특정 문자(열) 삽입

```python
print(",".join('abcd')) ➝ a,b,c,d
```

#### 📌 upper() / lower() / capitalize() / title()
- upper() : 소문자를 대문자로 변환
- lower() : 대문자를 소문자로 변환
- capitalize() : 문자열의 첫 번째 문자를 대문자로 변환
- title() : 각 단어의 첫 글자를 대문자로 변경

#### 📌 strip() / lstrip() / rstrip()
- strip() : 문자열의 양 옆 공백 제거
- lstrip() : 문자열의 왼쪽 공백 제거
- rstrip() : 문자열의 오른쪽 공백 제거

## 📂 문자열 포맷팅(Formatting)

#### 📌 포맷 코드 사용 : %s %d %f %c

#### 📌 format() 함수 사용
- 숫자 바로 대입하기

```python
print("I eat {0} apples".format(3))
```

- 문자열 바로 대입하기

```python
print("I eat {0} apples".format("five"))
```

- 숫자 값을 가진 변수로 대입하기

```python
number = 3

print("I eat {0} apples".format(number))
```

- 2개 이상의 값 넣기

```python
number = 10
day = "three"

print("I eat {0} apples. I was sick for {1} days.".format(number, day))
```

- 이름으로 넣기

```python
print("I eat {number} apples. I was sick for {day} days.".format(number=10, day=3))
```

- 인덱스와 이름을 혼용해서 넣기

```python
print("I eat {0} apples. I was sick for {day} days."format(10, day=3))
```

## 📂 실수 출력

#### 📌 형식 : "{위치 인덱스:포맷코드}".format(변수)

```python
pi = 3.141592

print("{0:.2f}".format(pi)) ➝ 3.14
print("{0:10.2f}".format(pi)) ➝ ______3.14
```

## 📂 문자열 정렬

#### 📌 왼쪽 정렬 '<'

```python
print("{0:<10}".format("hi")) ➝ hi________ ※ 밑줄( _ )는 공백을 의미
```

#### 📌 오른쪽 정렬 '>'

```python
print("{0:>10}".format("hi")) ➝ ________hi
```

#### 📌 가운데 정렬 '^'

```python
print("{0:^10}".format("hi")) ➝ ____hi____
```

#### 📌 공백 채우기

```python
print("{0:=^10}".format("hi")) ➝ ====hi====
print("{0:!<10}".format("hi")) ➝ hi!!!!!!!!
```

## 📂 날짜 출력 포맷

#### 📌 모듈 사용
- 날짜와 시간을 출력하기 위해서는 datetime 모듈을 선언해야한다
- ex) from datetime import date, datetime

#### 📌 예제

```python
from datetime import date, datetime, timedelta

today = date.today()
year = today.year
month = today.month
day = today.day

print("오늘 날짜 : {0}".format(today)) ➝ 오늘 날짜 : 2022-07-08
print("연도 : {0}".format(year)) ➝ 연도 : 2022
print("월 : {0}".format(month)) ➝ 월 : 7
print("일 : {0}".format(day)) ➝ 일 : 8
```

```python
from datetime import date, datetime, timedelta

currentTime = datetime.now().time()
hour = currentTime.hour
minute = currentTime.minute
second = currentTime.second
microsecond = currentTime.microsecond

print("현재 시간 : {0}".format(currentTime)) ➝ 현재 시간 : 01:02:39.220472
print("시 : {0}".format(hour)) ➝ 시 : 1
print("분 : {0}".format(minute)) ➝ 분 : 2
print("초 : {0}".format(second)) ➝ 초 : 39
print("마이크로초 : {0}.format(microsecond)) ➝ 마이크로초 : 220472
```

```python
from datetime import date, datetime, timedelta

today = date.today()

print("오늘 : {0}".format(today)) ➝ 오늘 : 2022-07-08
print("어제 : {0}".format(today + timedelta(days=-1))) ➝ 어제 : 2022-07-07
print("내일 : {0}".format(today + timedelta(days=1))) ➝ 내일 : 2022-07-09
```

```python
from datetime import date, datetime, timedelta

currentTime = datetime.now()
print("현재 시간 : {0}".format(currentTime)) ➝ 현재 시간 : 2022-07-08 01:02:39.221722

currentHour = currentTime + timedelta(hours=-1)
print("1시간 전 : {0}".format(currentHour)) ➝ 1시간 전 : 2022-07-08 00:02:39.221722
print("1시간 전 : {0}".format(currentHour.time())) ➝ 1시간 전 : 00:02:39.22172

print("1일 2시간 후 : {0}".format(currentTime + timedelta(days=1, hours=2))) ➝ 1일 2시간 후 : 2022-07-08 03:02:39.221722
```
