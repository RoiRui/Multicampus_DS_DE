## 📂 연산자(Operators)

#### 📌 산술 연산자

| 연산자 |   의미   |
| :----: | :------: |
|   +    |   덧셈   |
|   -    |   뺄셈   |
|   *    |   곱셈   |
|   /    |  나눗셈  |
|   //   |    몫    |
|   %    |  나머지  |
|   **   | 거듭제곱 |

#### 📌 산술 연산자의 우선순위

| 우선순위 |           연산자           |
| :------: | :------------------------: |
|    1     |          괄호 ()           |
|    2     |        거듭제곱 **         |
|    3     | 곱셈 *, 나눗셈 /, 나머지 % |
|    4     |       덧셈 +, 뺄셈 -       |

#### 📌 누적 대입 연산자

| 연산자 |   의미   |
| :----: | :------: |
|   +=   |   덧셈   |
|   -=   |   뺄셈   |
|   *=   |   곱셈   |
|   /=   |  나눗셈  |
|  //=   |    몫    |
|   %=   |  나머지  |
|  **=   | 거듭제곱 |

#### 📌 관계 연산자
- 두 값을 비교하여 결과를 반환한다 (결과는 True, False)

| 연산자 |    의미     |
| :----: | :---------: |
|   ==   |    같다     |
|   !=   |  같지 않다  |
|   >    |    크다     |
|   <    |    작다     |
|   >=   | 크거나 같다 |
|   <=   | 작거나 같다 |

#### 📌 논리 연산자

| 연산자 |      의미       |
| :----: | :-------------: |
|  and   | 논리곱 (그리고) |
|   or   |  논리합 (또는)  |
|   ot   |  부정 (아니다)  |

#### 📌 비트 연산자
- 2진수 연산

|          연산자           |                  의미                   |
| :-----------------------: | :-------------------------------------: |
|      & (Binary AND)       |   대응하는 2진수가 모두 1이면 1 반환    |
|      \| (Binary OR)       | 대응하는 2진수 중 하나라도 1이면 1 반환 |
|      ^ (Binary XOR)       |     대응하는 2진수가 다르면 1 반환      |
|      ~ (Binary NOT)       |          1이면 0, 0이면 1 반환          |
| \<\< (Binary Left Shift)  |  지정한 수 만큼 2진수를 왼쪽으로 이동   |
| \>\> (Binary Right Shift) |  지정한 수만큼 2진수를 오른쪽으로 이동  |
