## 📂 if 문

#### 📌 if 문이란?
- 조건식이 참이면 주어진 문장을 수행
- 조건식이 거짓이면 아무것도 수행하지 않는다

#### 📌 형식

```python
if(조건식):
  참일 경우 수행되는 문장
```

## 📂 if ~ else 문

#### 📌 if ~ else 문이란?
- 조건식이 참이면 if 문 수행
- 조건식이 거짓이면 else 문 수행

#### 📌 형식

```python
if(조건식):
  참일 경우 수행되는 문장
else:
  거짓일 경우 수행되는 문장
```

## 📂 if ~ elif ~ else 문

#### 📌 if ~ elif ~ else 문이란?
- 또 다른 조건이 있을 경우 elif 사용

#### 📌 형식

```python
if(조건식1):
  조건식1이 참일 경우 수행되는 문장
elif(조건식2):
  조건식2가 참일 경우 수행되는 문장
else:
  모든 조건식이 거짓일 경우 수행되는 문장
```

## 📂 중첩 if 문

#### 📌 중첩 if 문이란?
- if 문에 다른 if 문 포함

#### 📌 형식

```python
if(조건식1):
  if(조건식2):
    문장A
  else:
    문장B
else:
  문장C
```

## 📂 난수 생성

#### 📌 randint() 함수
- 최소부터 최대 사이의 임의의 정수 반환
- 형식 : randint(최소, 최대)

```python
from random import randint ※ Python에서 난수를 사용하기 위해서는 기본적으로 제공되는 random 모듈 사용

n = randint(1, 100)
```
