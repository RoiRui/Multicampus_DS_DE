## 📂 자료형(Data Type)

#### 📌 파이썬이 처리하는 자료형
- 정수(int) : -1, 0, 10, 100, ...
- 실수(float) : 3.14, -1.23, ...
- 문자열(str) : "홍길동", '123', 'A', ...
- 불(bool) : True, False

#### 📌 가변 자료형
- 파이썬에서의 변수는 지정된 자료형이 없다
- **저장한 값에 따라 변수의 자료형이 정해진다**

#### 📌 형변환 함수
- int(문자열) : 문자열을 정수로 변환
- float(문자열) : 문자열을 실수로 변환
- str(정수 또는 실수) : 정수 또는 실수를 문자열로 변환

```python
print("나는 현재 " + 23 + "살입니다") ➝ ERROR
print("나는 현재 " + str(23) + "살입니다") ➝ OK
```

```python
num = input("숫자 입력 : ")

print(num + 100) ➝ ERROR
print(int(num) + 100) ➝ OK
```

```python
x = input("실수 입력 : ") ➝ 12.5

y = x * 3
print(y) ➝ 12.512.512.5

y = float(x) * 3
print(y) ➝ 37.5
```

#### 📌 input() 함수
- 키보드로부터 입력 받은 값 반환
- input() 함수로부터 반환되는 값은 문자열이다
- 형식 : input("프롬프트 문자열")

```python
x = input("숫자1 입력 : ") ➝ 10
y = input("숫자2 입력 : ") ➝ 20
sum = x + y

print("합 : " + sum) ➝ 1020
```

```python
x = int(input("숫자1 입력 : ")) ➝ 10
y = int(input("숫자2 입력 : ")) ➝ 20
sum = x + y

print("합 : " + sum) ➝ 30
```

#### 📌 eval() 함수
- 문자열을 숫자로 변환하는 함수

```python
x = eval(input("숫자1 입력 : ")) ➝ 10
y = eval(input("숫자2 입력 : ")) ➝ 20
sum = x + y

print("합 : " + sum) ➝ 30
```

#### 📌 int() vs eval()

- int()는 input() 입력을 받을 시 소수점이 있으면 에러가 발생한다
- eval()은 input() 입력을 받을 시 정수, 실수 관계없이 모두 가능하다
