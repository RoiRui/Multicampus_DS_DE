## 📂 리스트

#### 📌 리스트란?
- 동일한 이름을 갖는 원소들의 연속 저장 영역
- 여러 개의 데이터가 저장되어 있는 장소(집합적 자료형)
- 각 원소는 인덱스(Index)로 구분

#### 📌 형식
- 리스트 생성 시 대괄호 [] 사용
- 리스트 = \[값1, 값2, ...\]

#### 📌 특징
- 리스트의 크기는 **가변적**이다 ➝ 원소의 개수에 따라 크기가 정해진다
- **다양한 종류의 데이터**를 하나의 리스트에 저장할 수 있다

## 📂 리스트 인덱싱(Indexing)

#### 📌 리스트 인덱싱이란?
- 리스트에서 인덱스 연산자를 통하여 요소를 참조(접근)하는 것

#### 📌 예시

```python
a = [1, 2, 3, 4, 5]
a[0] = 1, a[1] = 2, a[4] = 5, a[-1] = 5

b = [1, 2, 3, [10, 20]]
b[-1][0] = 10 b[3] = [10, 20]
```

## 📂 리스트 슬라이싱(Slicing)

#### 📌 리스트 슬라이싱이란?
- 리스트 안에서 범위를 지정하여서 원하는 요소들을 선택하는 연산

#### 📌 형식
- 리스트\[start, end\] : start에서 end-1 요소까지 선택

## 📂 리스트 연산

#### 📌 종류
- 리스트 합치기 : '+' 연산자 사용
- 리스트 곱하기 : '*' 연산자 사용
- 리스트 내용 변경 : 특정 위치의 요소에 값 저장

## 📂 리스트 복사

#### 📌 리스트 저장 원리
- 리스트 객체를 직접 저장하고 있는 것이 아니라 리스트 자체는 다른 곳에 저장되어 있고 **참조 값(Reference)만 변수에 저장**

<img width = 300, height = 200 src = "https://user-images.githubusercontent.com/106376150/178025816-f3541ad5-9da1-48ae-92b3-1cc18565c3ba.jpg">

#### 📌 얕은 복사(shallow copy)
- 실제 리스트는 복사되지 않고 참조 값(주소)만 복사
- 복사본 리스트 요소의 값을 변경하면 원본 리스트 요소의 값도 변경된다

```python
scores = [10, 20, 30, 40]
values = scores
```

<img width = 300, height = 200 src = "https://user-images.githubusercontent.com/106376150/178025828-59332bc7-d131-4dbd-b73a-d5f8f4f01948.jpg">

#### 📌 깊은 복사(deep copy)
- '=' 연산자만으로는 깊은 복사가 되지 않음
- list() 또는 deepcopy() 함수 사용
- 리스트의 값의 복사본을 새로 생성하여 반환
- 깊은 복사 시에 원본 리스트의 원소 값은 변경되지 않는다

## 📂 리스트의 주요 메소드(함수)

#### 📌 len()
- 리스트의 길이 반환(원소의 개수)

```python
a = [1, 2, 3, 4, 5]

print(len(a)) ➝ 5
```

#### 📌 count()
- 리스트 내에서 특정 요소의 개수 세기

```python
a = [1, 2, 3, 3, 5]

print(a.count(3)) ➝ 2
```

#### 📌 append()
- 리스트의 끝에 새로운 요소 추가
- 형식 : 리스트.append(값)

#### 📌 insert()
- 리스트의 특정 위치에 요소 삽입
- 형식 : 리스트.insert(위치, 값)

#### 📌 remove()
- 리스트에서 값에 해당하는 요소 제거
- 동일한 값이 여러 개 있는 경우 첫 번째 값만 제거 ➝ 다 제거하려면 for문 활용
- 형식 : 리스트.remove(값)

#### 📌 pop()
- 형식1 : 리스트.pop() ➝ 리스트의 마지막 요소 반환하고 삭제
- 형식2 : 리스트.pop(인덱스) ➝  인덱스 위치에 있는 요소 반환하고 삭제

#### 📌 extend()
- 리스트 확장
- 형식 : 리스트.extend()
- append() / insert() vs extend()

```python
append() / insert()
a = [1, 2, 3]
a.append/insert([4,5])
print(a) ➝ [1, 2, 3, [4, 5]]

extend()
a = [1, 2, 3]
a.extend([4, 5])
print(a) ➝ [1, 2, 3, 4, 5]
```

#### 📌 sort()
- 리스트의 원소들 정렬
- 원본 리스트 변경
- 형식1 : 리스트.sort() ➝ 오름차순 정렬
- 형식2 : 리스트.sort(reverse = True) ➝ 내림차순 정렬
- 형식3 : 리스트.sort(key = str.lower) ➝  대소문자 구별 X

#### 📌 reverse()
- 역순으로 원본 리스트 변경
- 형식 : 리스트.reverse()

#### 📌 max() / min()
- 리스트 내에서 최대/최소값 원소 반환
- 형식 : max(리스트) / min(리스트)

#### 📌 index()
- 리스트 안에서 원소의 위치 값 반환
- 원소가 존재하지 않으면 에러 발생
- 형식 : 리스트.index(값)

```python
a = [1, 2, 3, 4, 5]

print(a.index(3)) ➝ 2
print(a.index(10)) ➝ ERROR
```

#### 📌 in / not in
- 리스트 내에서 원소의 존재 여부 반환

```python
num = [1, 2, 3, 4, 5]

print(2 in num) ➝ True
print(3 not in num) ➝ False
```
