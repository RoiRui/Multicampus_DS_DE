## 📂 예외 처리(Exception Handling)

#### 📌 에러(Error)
- 문법적 오류나 실행 시간에 잘못된 메모리 접근 오류, 논리 오류, 사용자의 잘못된 입력 오류 등
- 오류가 발생하면 프로그램이 중단되고 에러 메세지가 출력된다
- ex) ZeroDivisionError : 0으로 나눈 경우
- ex) FileNotFoundError : File을 못 찾았을 경우
- ex) IndexError : 리스트 등에서 인덱스가 잘못된 경우

#### 📌 예외(Exception)
- 오류에 대처하기 위한 오류
- 예외는 주로 실행 중에 발생한다

#### 📌 예외 처리(Exception Handling)
- 오류 발생 시 프로그램이 중단되괴 에러 메세지가 출력되지 않도록 예외를 발생시켜 예외처리를 통해 정상 실행되도록 하기 위한 방법
- 예외 처리 방법 : try ... except 문 사용

```python
try:
  예외 발생 가능 문장
except 예외 유형:
  예외 처리 문장
[else:]
  예외 미발생 시 수행 문장
[finally:]
  예외 유무와 상관 없이 항상 수행되는 문장
```

```python
try:
  print(10/0)
except ZeroDivisionError:
  print("0으로 나눌 수 없습니다")
```

```python
try:
  print(10/0)
except ZeroDivisionError as e:
  print("0으로 나눌 수 없습니다 : ", e) ➝ 0으로 나눌 수 없습니다 : division by zero
```

```python
a = [1, 2, 3]

try:
  print(10/0) ※ 첫 번째 에러만 처리된다
  print(a[4])
except ZeroDivisionError as e:
  print("0으로 나눌 수 없습니다 : ", e) ➝ 0으로 나눌 수 없습니다 : division by zero
except IndexError as e:
  print("인덱스 범위를 벗어났습니다 : ", e)
  
try:
  print(a[4]) ※ 첫 번째 에러만 처리된다
  print(10/0)  
except ZeroDivisionError as e:
  print("0으로 나눌 수 없습니다 : ", e)
except IndexError as e:
  print("인덱스 범위를 벗어났습니다 : ", e) ➝ 인덱스 범위를 벗어났습니다 : list index out of range
```

```python
try:
  num = int(input("숫자 입력 : "))
except ValueError:
  print("숫자가 아닙니다") ➝ ValueError가 발생한 경우 '숫자가 아닙니다' 출력
else:
  print(num) ➝ ValueError가 발생하지 않은 경우 입력받은 num 값을 출력
```

```python
try:
  f = open("test.txt", 'r')
except FileNotFoundError:
  pass ➝ FileNotFoundError 발생 시 아무것도 하지 않고 그냥 넘어간다
else:
  data = f.read()
  print(data) ➝ 예외가 발생하지 않는다면 test.txt의 내용을 출력
  f.close()
finally:
  print("종료") ➝ 예외 발생 여부에 상관없이 '종료' 출력
```
