## 📂 함수(Function)

#### 📌 함수란?
- 특정 작업을 수행하는 코드 집합

#### 📌 함수를 사용하는 이유?
- 코드 재사용 / 중복된 코드 제거 / 작업 분해 ➝ 경제적 / 프로그램 가독성 증가 / 프로그램 유지 관리 용이

#### 📌 함수의 종류
- 내장 함수
  - 파이썬에 이미 만들어져있는 함수들
  - 형식에 맞춰 함수 이름만 호출해서 사용
- 사용자 정의 함수
  - 사용자가 임의로 만들어 사용하는 함수

#### 📌 함수의 구조

```python
def 함수명(매개변수1, 매개변수2, ...):
    수행 문장
    반환값

※ def? 함수 정의 키워드
※ 매개변수? 함수로 전달되는 값을 받는 변수(쌩략가능)
※ 반환값? 함수가 특정 기능을 수행하고 돌려주는 결과 값(return 문 사용, 함수가 호출된 곳으로 반환)
           리스트, 딕셔너리도 return 가능하며 반환값이 없는 경우 None 출력
```

#### 📌 함수 명명 규칙
- 함수의 목적을 설명하는 동사 · 명사 사용

#### 📌 함수 사용 방법
- 함수를 사용하려면 함수 이름을 이용해 호출(Call)해야한다
- 함수를 호출하기 전에 먼저 함수를 정의해야한다

```python

    num1 = int(input("숫자 1 입력 : "))
    num2 = int(input("숫자 2 입력 : "))
    return num1 + num2

```

#### 📌 매개변수(Parameter) · 인수(Argument)
- 매개변수 : 함수로 전달되는 값을 받는 변수
- 인수 : 함수에게 실제로 전달되는 값
- 마찬가지로 리스트와 딕셔너리 전달 가능

```python
def get_average(k, e, m)
    return (k + e + m) / 3

kor = int(input("국어 점수 입력 : "))
eng = int(input("영어 점수 입력 : "))
math = int(input("수학 점수 입력 : "))
print("평균 : %.2f" % get_average(kor, eng, math))
```

#### 📌 위치 인수 vs 키워드 인수
- 위치 인수
  - 함수 호출 시 위치에 의하여 구별하는 방식
  - 매개변수의 순서대로 인수를 전달하여 사용하는 경우

```python
add(a, b, c) ←→ Call : add(10, 20, 30) ※ 순서대로 a는 10, b는 20, c는 30과 매치
```

- 키워드 인수
  - 인수들 앞에 키워드를 두어서 인수를 구별하는 방식
  - 인수의 위치가 매개변수의 위치와 달라도 된다

```python
add(a, b, c) ←→ Call : add(c = 30, a = 10, b = 20) ※ 위치 인수와 달리 변수명을 이용해 매치
```

- **위치 인수와 키워드 인수를 섞어서 사용해도 되지만 위치 인수를 키워드 인수보다 앞에 두어야한다**<br>

```python
Call : add(10, c = 30 , 20) ➝ O
Call : add(c = 30, 20, 10) ➝ X
```


## 📂 지역변수와 전역변수
- 지역변수(Local Variable)?
  - 함수 내부에서 정의된 변수
  - 함수 안에서만 사용가능
  - 함수 호출 시 생성되고 함수가 종료되면 소멸되어 더 이상 사용할 수 없다
  - 함수의 매개변수도 지역 변수 ➝ 외부로부터 값을 전달 받아서 함수 내부에서만 사용하는 지역변수
- 전역변수(Global Variable)?
  - 함수 외부에서 정의된 변수
  - 프로그램 내 모든 곳에서 사용 가능
  - 함수 내에서 전역 변수의 값을 변경하려면 global 키워드 사용
